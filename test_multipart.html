🎤 TTS Multipart Response 테스트

1. 간단한 테스트 (파라미터 없음)

2. 커스텀 JSON 테스트

3. cURL 명령어 예시

# 간단한 테스트 curl -X POST "http://localhost:8003/tts/test-multipart" \ -H "accept: */*" \ --output response.txt # 커스텀 JSON 테스트 curl -X POST "http://localhost:8003/tts/convert-rag-response" \ -H "accept: multipart/form-data" \ -H "Content-Type: application/json" \ -d '{ "success": true, "messages": [ {"HumanMessage": "테스트 질문"}, {"AIMessage": "테스트 답변입니다"} ], "citations": [] }' \ --output response.txt
